## Hazer DMX
DMX Control for Antari HZ-100 Hazer  

Operation:  
Turn on Hazer output by setting control to VCC 

Makes use of DMX Serial Library  
https://github.com/mathertel/DMXSerial

### Hazer TRS Wiring  
1/4" TRS Control Cable  
T - Vcc 8v  
R - Control  
S - Ground
